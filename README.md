# (WIP) pt.md
Library for working with https://pt.md

## Prerequisites

- [ ] NodeJS >= v8.x.x

## Installation

```javascript
npm install pt.md
```

## Usage

Running tests:

```javascript
USER='' PASS='' npm test
```

Running linter (eslint):

```javascript
npm run lint
```

Importing library

```javascript
Pt = require('pt.md');
```
